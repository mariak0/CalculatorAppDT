<h3>Simple Calculator App with Login Function</h3>

This is an **Android app** that includes a login screen and a calculator.

<h4>Features</h4>
<ul>
<li>Login with username and password validation</li>
<li>Shows error messages if input is wrong</li>
<li>Toast message when login is succesful</li>
<li>Moves to next activity using Intent and sends the username</li>
</ul>

<h4>Calculator</h4>
<ul>
<li>Numbers from 0–9</li>
<li>Operators: +, -, *, /</li>
<li>Supports multiple operations (like 2+2+5*2)</li>
<li>Shows what user types on the screen</li>
<li>Calculates result with correct priority</li>
</ul>

<h4>Extra Features</h4>
<ol>
<li>Delete button (removes one character)</li>
<li>CE button (clear all text)</li>
<li>Handles division by zero</li>
</ol>
