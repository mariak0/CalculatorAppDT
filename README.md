Simple Calculator App with Login Function

This is an Android app made in Java that includes a login screen and a calculator.

Features
Login with username and password validation
Shows error messages if input is wrong
Toast message when login is succesful
Moves to next activity using Intent and sends the username
Calculator
Numbers from 0–9
Operators: +, -, *, /
Supports multiple operations (like 2+2+5*2)
Shows what user types on the screen
Calculates result with correct priority
Extra
Delete button (removes one character)
CE button (clear all text)
Handles division by zero
Tech
Java
Android Studio
XML layouts
Notes

This app was created for learning purposes and practise Android basics, within the PMSc Digital Transformation.
